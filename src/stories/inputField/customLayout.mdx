It is possible to change structure or the layout of the Input field as below.

<Source
  language="tsx"
  code={`import {InputField} from '@metalnet/baremetal/input';`}
/>

And change structure as the way you want.

<Source
  language="tsx"
  code={`<InputField size={"lg"}>
  <div className="flex items-center">
    <InputField.Label htmlFor={id} className="mr-1" size={"sm"}>
      {label}
    </InputField.Label>
    <InputField.Input id={id} type={type} onChange={onChange} />
  </div>
  <InputField.Error size={"sm"} className="ml-10 mt-2">
    {typeof error === "string" && error}

    {Array.isArray(error) && (
      <ul>
        {error.map((err) => (
          <li key={err}>{err}</li>
        ))}
      </ul>
    )}

  </InputField.Error>
</InputField>`}

/>

<Primary />
<Controls />
```
