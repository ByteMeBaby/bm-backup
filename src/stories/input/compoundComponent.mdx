You can customize the structure of the input very easily with compound components.

<Source
  language="tsx"
  code={`import {InputField} from '@metalnet/baremetal/input';`}
/>

And use structure it the way you want

<Source
  language="tsx"
  code={`<InputField>
  <InputField.Label htmlFor={id}>{label}</InputField.Label>
  <InputField.Input id={id} type={type} />
  <InputField.Error>error</InputField.Error>
</InputField>`}
/>

<Primary />
```
